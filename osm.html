Attempt 4: Ontario Street Map (OSM)

Ontario Street Map is a free, open-source web map maintained by the public. We tested out some capabilities of OSM to see if it is a suitable solution for our project. Here's what we found...

Explore OSM

  1. Create an OSM account: https://www.openstreetmap.org/
  2. Complete the Guided Tour
  3. Review the results

TAKE AWAYS

Advantages Disadvantages
Collaborative, any changes you make go live for everyone when saved The imagery of the basemap changes depending on the zoom
Very user-friendly The buildings and roads don't completely line up, which can be problematic for transit routing
Can edit the map in-browser, without having to download anything Cannot do further analysis through OSM, just adding and editing features
The data exported isn't just a simple shapefile, more conversions required
The transit data for Lindsay in OSM is not up-to-date

It was decided that this was not a suitable web solution for the issue we are trying to solve.
