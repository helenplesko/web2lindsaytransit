Public Transit Routing in Lindsay, Ontario

Problem Statement

Ideal

In an ideal world, people living and visiting Lindsay, Ontario, would have access to a user-friendly public web application that will help them navigate around the city via public transit.

Reality

At the moment, members of this rural community do not have quick access to public transit routes on common navigation apps like Google Maps. They must rely on prior knowledge, word-of-mouth, or static maps to help them reach their destination instead.

Consequences

If a solution to this is not found, the user may end up using out-of-date transit route information, get lost, miss their bus, or take a bus heading in a different direction.

Proposal

To solve the problem, AHRS Transportation is exploring various web technologies and finding the best one that will help make a user-friendly web application that will help the general public navigate rural areas like Lindsay using public transit.

Technology Explored

ArcGIS Experience Builder, ArcGIS Dashboard, ArcGIS SDK, Open Source (Leaflet, OSM, Mapbox)
