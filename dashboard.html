Attempt 1: ArcGIS Dashboards

We tried to solve the problem using ArcGIS Dashboards.

What is Dashboard?

ArcGIS Dashboards conveys information by presenting a location based analytics and visualize interactive data on a single screen.ArcGIS Dashboards help in making decisions,visualizing trends and monitor status in real time . Dasboards are one of the most essential information products like apps contributing a significant part in geospatial infrastructure.

How did ArcGIS Dashboard helped us to find the solution?

During the initial stage of problem resolving our team members discussed together that how the Dashboard can help us in building up a web application to for Lindsay transit.As Dashboard is one of the main contributor for accessing infrastructure building a transit app in dashboard was considered as a good option.There were some challenges and on the other hand there were also good outcomes from Dashboard that helped to find the ultimate solution.Some of our point of interests and challenges are given below.We started investigating about Dashboard in ArcGIS online and refered several dashboards in ArcGIS online to acquire some idea about it.The main steps that we followed to create a dashboard are given below.

Steps/Procedure we followed

  • Login to ArcGIS Online - User should have an account in Arcgis Online
  • Go to ArcGIS Dashboard in ArcGIS Online , Click "Create Dashboard"
  • Added the Map of Lindsay transit showing red, blue, green and orange bus routes and bus stops.
  • In the Settings menu the scale bar settings were given and set the theme to dark or white.
  • Added Splash screen to nofify and display the logo and a quick overview about the app. The element is modified and added to the final solution.
  • Added four Lists in Dashboard displaying the name of the busstops and the corresponding time that the bus will reach in bus stop with a serach button.We tried to add a selection query in the dashboard but it didn't work so well.
  • Then we added indicators
  • Added a table conatining the transit fares.

Challenges faced

One of the main challenge was that we were not able to add a selection query to select the bus and bus route from the list element. It became less interactive when viewed from the point of technology.
