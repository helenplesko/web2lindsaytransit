Attempt 2: SDK

We tried to solve the problem using ArcGIS SDK. Here's what we found:

Requirements: ArcGIS Developers or ArcGIS Online Accunt to access the dashboard and API Key.

API key can be accessed by creating ArcGIS Developers account for free.

  1. Visit ArcGIS Developers at: https://developers.arcgis.com/
  2. Click on the Start building for free( if you do not have account)or Sign In
  3. Within Dashboard/ API keys create new API key and add allowed Referrers where that API keys can be used for security reasons

Step 2: Explore Quick Start Guide Tutorial

  1. Access the tutorial code at: https://developers.arcgis.com/javascript/latest/tutorials/find-a-route-and-directions/
  2. Copy the code using the sandbox at :https://developers.arcgis.com/javascript/latest/sample-code/sandbox/?sample=route
