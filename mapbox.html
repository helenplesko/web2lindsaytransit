Attempt 5: Mapbox

Mapbox is a mapping and location platform for creating and customizing maps. It uses APIs and SDKs and is used by many reputable companies. Accounts are free, but many capabilities are not. We tested out some capabilities of Mapbox to see if it is a suitable solution for our project. Here's what we found...

Explore Mapbox

  1. Create a Mapbox account
  2. Complete the 'Getting started with Mapbox Standard in Studio,' which can be found here: https://docs.mapbox.com/help/tutorials/aa-standard-in-studio/
  3. Review the results

TAKE AWAYS

Advantages Disadvantages
The user interface has a "crisp look and feel" Many of the capabilities require coding
Pricing is flexible, free tier to start There isn't documentation on everything
Super customizable, using API and SDK No ability for field data collection
Has routing and geocoding processes Not completely open source as people believe
Tutorials available to follow Need to pay to use the services if we can't get free licensing for the student account
Has an option to view the routes with 3D buildings, which can make travelling easier, as you can locate landmarks as you go In the free account, you have 50,000 free map loads for web - I am assuming we will go beyond this, especially when the map is made available to the public

It was decided that this was not a suitable web solution for the issue we are trying to solve.
