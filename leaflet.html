Attempt 3: Leaflet

Leaflet is a free, open-source JavaScript library used to create interactive mobile solutions. We tested out some capabilities of Leaflet to see if it is a suitable solution for our project. Here's what we found...

Step 1: Download Leaflet

Click the button below to visit the Leaflet homepage. Click on the latest stable version of Leaflet (in our case, Leaflet 1.9.4). Then unzip the download folder and store in your website's directory.

Visit Leaflet

Step 2: Complete the Quick Start Guide Tutorial

Click the button below to access the tutorial code.

Tutorial Code

Change the coordinates of the polygon by copying and pasting the coordinates from Google Maps. Then make other customizations to the code, including changing the map center, changing the coordinates of the pop up and polygon, changing the pop up text, adding a title, and changing the map size.

Step 3: Review the results of the tutorial

View Results

Step 4: Import geojson file to Leaflet

Click the button below to navigate to the online converter used to convert the Lindsay bus route and bus stop shapefiles provided by GeoHub to geojson.

MyGeodata Converter

Copy and paste the geojson code into the Leaflet html page and make other customizations, including changing the zoom extent and removing the pop up. Commit changes.

Plethora of JavaScript properties organized by type/usage Need to do the extra step of converting data to geojson
