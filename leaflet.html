Attempt 3: Leaflet

Leaflet is a free, open-source JavaScript library used to create interactive mobile solutions. We tested out some capabilities of Leaflet to see if it is a suitable solution for our project. Here's what we found...

Step 1: Download Leaflet

  1. Visit Leaflet at: https://leafletjs.com/download.html
  2. Click on the latest stable version of Leaflet (in my case, Leaflet 1.9.4)
  3. Unzip the downloaded folder and store in your website's directory

Step 2: Explore Quick Start Guide Tutorial

  1. Access the tutorial code at: https://leafletjs.com/examples/quick-start/
  2. Change the coordinates of the polygon by copying and pasting coordinates from Google Maps
  3. Make other customizations to the code, including:
    • change the map center
    • change the coordinates of the pop up
    • change the coordinates of the polygon
    • change the pop up text
    • add a title
    • change the map size

Step 3: Review the results of the tutorial

Step 4: Import geojson file to Leaflet

  1. Use an online converter to convert the Lindsay bus route and bus stop shapefiles provided by Ontario GeoHub to geojson
  2. Copy and paste the geojson code into the Leaflet code and commit changes
  3. Make other customizations to the code, including:
    • change zoom extent
    • remove pop up

Step 5: Review the results of geojson in Leaflet

TAKE AWAYS

Advantages Disadvantages
Free, open source solution Needs lots of customization, which can be time consuming
Compatible with mobile phones Uses programming languages, which not everyone is comfortable with
Plethora of JavaScript properties organized by type/usage Need to do the extra step of converting data to geojson
Tutorials available
